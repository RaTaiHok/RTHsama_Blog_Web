# 关于咱本身

名字叫做RTH猫雨音！一只粉毛猫猫~，是一只音游/Minecraft/代码萌新猫，关于这个名字，起源于小学时起的一个很中二的中文名字，后来觉得太尴尬了，就起了一个英文名，简写就是RTH，到高二的时候，基于RTH起了一个RaTaiHok的笔名，直到高三才完善了名字的全称，叫RainToneH，由此起的中文名叫做雨音，后来延伸到日语的猫アメオン（Neko Ame On -> 猫雨音）
